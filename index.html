My Work

Instagram PhotoTracker

  • UI / UX
  • API Management
  • Processing (Java)

Making use of Instagram's API and created using Processing (a framework focused for designers built on Java), the Instagram PhotoTracker is a data visualization program which allows users to view recent posts around the downtown Boston area based on the location it was posted. Additionally, these posts can be narrowed down by searching for specific tags.

Music Editor

  • Software Design
  • MVC Framework
  • Java

Transforming text to audio, this program loads notes from a file and plays back the corresponding tune in real-time. The user can pause, rewind, or skip at any point during the playback of the song. Additionally, the user can add or delete notes from the track using the UI. This project was built using the MVC framework, and components can be swapped out at will!

nu:OnTrack

  • Responsive Design
  • UI / UX
  • HTML / CSS / JS

A redesign of Northeastern's current nuCareers site, this project utilizes mobile-first and responsive design practices to create a better user experience for students seeking co-ops. Additionally, it was designed to fit with Northeastern's current branding and themes while still providing a modern update.
