NU:OnTrack - HTML/CSS/JS

Created as a final project for my Interaction Design 1 course, NU:OnTrack is a take on Northeastern's Co-op search web portal that is revamped with responsive design as well as an improved user experience. The site keeps with the brand of Northeastern, and the mobile first perspective makes sure it transitions cross-platform.

Below details the task, thought-process, and features of the project, as well as providing links to the code on Github.

The Task at Hand

Through my Interaction Design: Responsivness course, I was tasked with creating a web page for an application, real or fake, that worked across all devices. After using Northeastern's Co-op search portal, NuCareers, I decided to try and revamp that site, since it is often a pain to use for many students.

The main changes I wanted to make were to make the site accessible on mobile devices, easy to navigate and find specific functionality, and to reorganize the hierarchy of the site in a way more intuitive to current students. Additionally I wanted to try and keep the site somewhat in-line with the Northeastern brand, since it would be a resource used by the University in theory.

The Design Process

Before designing the site, I started brainstorming the different use cases - creating personas to help understand user needs. The list I came up with had the following tasks as priorities: view new job listings, manage and review current submissions and documents, and applying to jobs quickly and easily.

After creating my initial list of percieved needs and core functionality, I developed wireframes to draft the navigation and design of the site. Once completed, I started user-testing to get feedback from actual students at Northeastern. This process was very enlightening and helped revise my designs to accommodate for what I hadn't thought of.

With feedback in hand, I began creating the digital designs directly with HTML and CSS. Throughout the process I kept my stylings clean, eliminating redundant declarations wherever necessary. Each section took multiple iterations to get to a point where I was pleased, but in the end came together to create a cohesive website.

Final Results

At the end of the process I came up with a revamped Northeastern Co-op search site, dubbed NU:OnTrack. The site was much easier to use than the current site, and held up across devices. While the direct functionality of the site wasn't implemented, I created mock actions such as uploading documents, logging in, and applying for co-op positions which helped showcase the potential user experience.

NU:OnTrack is certainly better than the exisitng solution of NuCareers, however I would like to revisit and update the project at a later date. The project was created during my second year, and I have learned much since then. My goals for future iterations would be to create real functionality, most likely achieved through PHP and MySQL or the MEAN stack. Until then, the current site is live on my Github, and accessible through the link below.
